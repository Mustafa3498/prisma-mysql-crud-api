# 🚀 Prisma + MySQL CRUD API

This is a simple REST API built using **Node.js, Express, Prisma ORM, and MySQL**.  
It demonstrates basic CRUD operations (Create, Read, Update, Delete).

---

## 📌 Tech Stack

- Node.js
- Express.js
- Prisma ORM
- MySQL (XAMPP / phpMyAdmin)
- Postman (for testing)
